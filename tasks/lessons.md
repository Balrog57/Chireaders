# Leçons Apprises (Memory)

## Concepts et Patterns
- **Stack Technique** : React Native (Expo) avec scraping via `react-native-cheerio`.
- **Navigation** : Utilisation de `@react-navigation/native` avec une structure Stack et Bottom Tabs.
- **Persistance** : `AsyncStorage` pour les favoris et l'historique de lecture.
- **Tâches de fond** : `expo-task-manager` et `expo-background-fetch` utilisés pour les notifications de mise à jour.
